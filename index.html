필요한 순간,
 SRT 좌석을 자동으로

원하는 시간대의 잔여 좌석을 빠르게 찾아 예약까지 이어주는 자동 예매 도우미입니다.

고급 설정 (헤드리스)
헤드리스 모드는 브라우저 창을 띄우지 않고 백그라운드에서 실행해요. 오류 상황을 확인하려면 끄고 진행하세요.
GitHub Pages에선 이 웹앱은 정적 호스팅만 제공돼요. 자동화를 위해서는 별도 백엔드가 필요합니다. 로컬에서 서버를 실행 중이면 http://localhost:3000을 입력하세요. 원격 서버가 있다면 해당 URL을 입력하세요.

안심하고 사용하세요

  • 이 웹앱은 아이디/비밀번호를 저장하지 않습니다.
  • 브라우저에 남는 정보는 역/날짜 같은 정보들 뿐이에요. 비밀번호는 절대 저장하지 않아요.
  • 자동화는 내 PC의 크롬을 열어 SRT 공식 사이트를 직접 조작해요. 화면에서 모든 과정을 확인할 수 있어요.
  • 언제든 중지를 눌러 즉시 멈출 수 있어요. 실행 기록은 화면의 로그에서만 확인돼요.
